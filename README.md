Aplicación de consola en Java para gestionar tareas con sistema de prioridades. 
Proyecto educativo que implementa conceptos fundamentales como entrada/salida 
con Scanner, estructuras condicionales (if-else), operadores lógicos y 
aritméticos, y validación de datos. El usuario puede agregar tareas asignándoles 
una prioridad del 1 al 5, y el sistema calcula la suma total de prioridades.

Características:
- Menú interactivo de consola
- Validación de prioridades (1-5)
- Acumulador de prioridades totales
- Manejo básico de entrada de datos
